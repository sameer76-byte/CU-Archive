Acceptable Formats

Learn which file formats we accept for submissions, size limits, and naming conventions

Primary Format: PDF

PDF (Portable Document Format) is our preferred and primary format for all question papers. PDF files ensure document integrity, cross-platform compatibility, and maintain consistent formatting across all devices.

PDF Requirements

Text-based PDFs

Status: Preferred

Created from digital sources (Word, LaTeX, etc.). These have searchable text and smaller file sizes.

Scanned PDFs

Status: Accepted

Created by scanning physical papers. Must be clear, legible, and properly oriented.

File Size Limit

Maximum: 10MB per file

Larger files may be rejected. Compress scans if needed while maintaining readability.

Additional Formats

For supporting materials, screenshots, or reference documents, we also accept these formats:

Format Status Purpose Maximum Size Notes
JPG/JPEG Accepted Screenshots, photos of papers 5MB High quality, clear images only
PNG Accepted Screenshots, diagrams, charts 5MB Lossless format for diagrams
DOC/DOCX Limited Use Reference materials only 5MB Not for question papers

Important Note

While we accept these additional formats for reference materials, question papers must be submitted as PDF files whenever possible. Other formats are for supplementary materials only.

Unacceptable Formats

The following formats will be rejected automatically. Please avoid submitting files in these formats.

Format Status Reason for Rejection Alternative
ZIP/RAR/7Z Not Accepted Archived files cannot be reviewed individually Submit individual files
EXE/BAT/MSI Not Accepted Security risk, executable files Convert to PDF
Password-protected files Not Accepted Cannot be reviewed or processed Remove password before submission
Corrupted files Not Accepted Cannot be opened or verified Check file integrity before submission

File Naming Convention

Properly naming your files helps us organize and categorize submissions efficiently. Follow this naming convention:

Format: Course_Subject_Year_Semester.pdf

Example: BSc_Mathematics_2022_3.pdf

Breakdown:

  • Course: BSc, BCom, BA, etc.
  • Subject: Mathematics, Computer Science, Economics
  • Year: 2022, 2021, 2020
  • Semester: 1, 2, 3, 4, 5, 6 (optional if not applicable)

Additional Examples

Good Examples

  • BCom_Accounting_2021_4.pdf
  • BA_History_2020.pdf (no semester)
  • BSc_Physics_2019_Final.pdf (paper type instead of semester)

Bad Examples

  • Document1.pdf (too vague)
  • scan0001.jpg (generic name)
  • my_paper_final_version_2_revised.pdf (too long, unclear)
